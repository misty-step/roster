A large library. A narrow launch.

Roster keeps reusable agent primitives separate from private composition, then resolves one named agent into exactly the context its role declares.

Source-qualified identities prevent accidental override. The manifest preserves every inclusion chain and digest.

Workspace isolation without a workspace taxonomy

The nearest config wins completely. A private R90 config can expose Habitat and R90 roles; the home Misty Step config can expose Powder. Workspace is not another field in every agent—the effective agent names and declarations tell the truth.

Small deterministic spine

Rust owns strict schemas, graph resolution, provenance, projection, redacted receipts, and process behavior. Models own semantic work. Roster does not route tasks, supervise runs, manage a board, or mutate global Harness state.
