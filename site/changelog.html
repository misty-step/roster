Release notes

Roster remains pre-1.0. These entries describe shipped product boundaries, not an invented stable history.

2026-07-15 · v0.2.1

Prove prefix-safe rollback

  • Corrected rollback guidance for custom installation prefixes and made the website install recipe use one explicit prefix throughout.
  • Added attested previous-version rollback and roll-forward replay to each public Linux and macOS cold start.
  • Published the redacted authenticated-dispatch and independent-review record from the first public release.

2026-07-15 · v0.2.0

Install without a development checkout

  • Added checksummed macOS and Linux archives that keep the native CLI and matching public plain-file library together.
  • Added automatic installed-source discovery, GitHub build provenance, clean-room install replay, explicit rollback, and Landmark-backed release evidence.

Exact composition at dispatch

  • Introduced strict public primitives, packs, roles, private agent bindings, source-qualified resolution, provenance manifests, and the interactive agent picker.
  • Added fail-closed Codex, Claude Code, and OMP projections, redacted receipts, rescue sessions, nearest-config replacement, and the optional operation-authority seam.
  • Retired workstation sync, doctor, installed agent trees, rich per-agent schemas, hooks, and the HTTP/MCP/UI services.
