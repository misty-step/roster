# Launch the nonprofit compliance package

Priority: P2 · Status: pending · Estimate: L

## Goal

Turn nonprofit expertise into a visible product for small local nonprofits:
990/199 preparation, board-facing status packet, compliance calendar, and
document readiness workflow.

## Children

1. Add nonprofit service positioning and route to `rkc-website`.
2. Define nonprofit checklist templates in the secure workflow backbone.
3. Build board packet generator with plain-English filing/compliance summary.
4. Add compliance calendar for federal, California, payroll, contractor, and
   charity-registration obligations.
5. Add restricted-fund and grant-reporting prompts.
6. Add internal-control checklist and issue escalation.
7. Add referral script for nonprofit boards, bookkeepers, and grant consultants.

## Notes

Target project: greenfield `rkc-nonprofit-desk`, with website and Doc Chaser
integration.

Why: Cecile's public positioning includes nonprofits; small nonprofits need
translation and board-facing clarity more than generic tax-prep copy.
