# Critique Shape

Lead with the highest-leverage issue. Avoid a laundry list.

```markdown
**Design Critique**
- Intent:
- Artifact inspected:
- Primary issue:
- Recommended direction:
- Specific changes:
- Verification needed:
```

Each finding names evidence from the artifact and one concrete change. If the
issue is only preference, say so; if it blocks comprehension or trust, say
that.
